# Step 9.3 - TrackEditor 軌道編輯器

## 建立的檔案

### src/ui/TrackEditor.ts

## 功能

| 操作 | 動作 |
|------|------|
| 左鍵點擊空白處 | 新增控制點 |
| 左鍵拖曳控制點 | 移動控制點 |
| 右鍵 | 移除最後一個控制點 |

## 方法

| 方法 | 說明 |
|------|------|
| `activate()` | 啟動編輯器 |
| `deactivate()` | 停用編輯器 |
| `draw()` | 繪製控制點和連接線 |
| `drawHint()` | 繪製編輯提示 |
| `getTrack()` | 取得自訂軌道 |
| `isValid()` | 檢查軌道是否有效 |
| `onChange(callback)` | 設定變更回呼 |
| `reset()` | 重置編輯器 |

## 視覺設計

- 起點：綠色圓點，標示「起」
- 終點：紅色圓點，標示「終」
- 中間點：藍色圓點
- 連接線：藍色虛線
