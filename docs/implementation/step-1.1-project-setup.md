# Step 1.1 - 專案基礎設定檔

## 建立的檔案

### package.json
- 專案名稱：marble-run
- 使用 ES modules
- 開發指令：`npm run dev`
- 建置指令：`npm run build`
- 依賴：typescript、vite

### tsconfig.json
- 目標：ES2020
- 嚴格模式啟用
- 支援 DOM 類型

### vite.config.ts
- base 設為 `/marble_run/`（對應 GitHub Pages 路徑）
- 輸出目錄：dist

## 下一步

執行 `npm install` 安裝依賴
