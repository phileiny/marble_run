# 再玩一次按鈕驗證

## 測試案例

### TC-1：預設軌道不顯示再玩一次

**步驟：**
1. 選擇預設軌道（經典斜坡）
2. 開始模擬
3. 等待模擬結束

**預期結果：**
- [x] 只顯示「重新選擇」按鈕
- [x] 不顯示「再玩一次」按鈕

---

### TC-2：自訂軌道顯示再玩一次

**步驟：**
1. 選擇「自訂軌道」
2. 繪製軌道（至少 2 個控制點）
3. 開始模擬
4. 等待模擬結束

**預期結果：**
- [x] 顯示「再玩一次」按鈕（綠色）
- [x] 顯示「重新選擇」按鈕

---

### TC-3：再玩一次功能

**步驟：**
1. 自訂軌道模擬結束後
2. 點擊「再玩一次」

**預期結果：**
- [x] 彈珠從起點重新開始
- [x] 軌道和收集盒保持不變
- [x] 滾動音效重新播放
- [x] 不會立即顯示失敗

---

### TC-4：重新選擇功能

**步驟：**
1. 自訂軌道模擬結束後
2. 點擊「重新選擇」

**預期結果：**
- [x] 回到軌道選擇畫面
- [x] 自訂軌道被清除
- [x] 可以重新選擇任何軌道

---

### TC-5：多次再玩一次

**步驟：**
1. 自訂軌道模擬結束
2. 點擊「再玩一次」
3. 再次模擬結束
4. 再點擊「再玩一次」

**預期結果：**
- [x] 可以無限次重新模擬
- [x] 每次都從起點開始
- [x] 不會出現錯誤

---

再玩一次功能通過所有測試案例。
